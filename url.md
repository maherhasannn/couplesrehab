# couplesrehab.com URL Map

## Site model (read first)
CouplesRehab.com presents as a **national addiction treatment placement and referral network**, established 2014. Active voice: "we," "our placement team," "we verify benefits," "we coordinate admission," "available 24/7." It coordinates admission into licensed providers but is **not itself a treatment facility**.

**The only conversion goal: drive phone calls to 888-500-2110.** Assessment is the secondary CTA. Canonical model page: `/couples-detox-rhode-island/`.

### Hard rules for every page (protects rankings + the call)
- Keep the **"placement/referral network, not a treatment facility"** disclosure in the opening and editorial disclaimer — this is what makes the active voice defensible. Never write "our facility admits" or claim to deliver treatment directly.
- **Never guarantee** coverage outcomes, in-network status, out-of-pocket figures, or a specific joint-placement configuration. Joint placement = "regularly possible, never guaranteed ahead of time"; coverage = "verified before any commitment."
- CTA language: "Call 888-500-2110," "call a care navigator," "begin the placement process," "verify your benefits," "Take the Couples Assessment." Keep the urgency.
- Every page touching overdose or severe withdrawal shows: call 911 for danger; call/text 988 for crisis (plus any state crisis line). Crisis blockquote near top AND at close.
- Match the RI page for clinical depth and page furniture (byline, crisis blockquotes, mid-page CTA banners, trusted-sources list, editorial disclaimer) — see `rehab-content-skill.md`.

## CTA modes
- **call-primary**: phone CTA above the fold + repeated; assessment secondary. (emergency-intent, geo, insurance)
- **assessment-primary**: assessment CTA leads, call secondary. (education, comparison, top-of-funnel couples)
- Both always include the (888) 500-2110 number.

## How to use this map (instruction for the writing skill)
When given a slug or topic:
1. Find its row below (or the closest cluster if it's a new topic).
2. Use **page type** to select the article template/structure.
3. Use **parent/pillar** + **cluster** to build internal links — always link up to the parent pillar, sideways to 2–3 cluster siblings, and to `/couples-assessment/` + `/crisis-support/`.
4. Use **CTA mode** to set conversion action (always the call).
5. Use **target keyword** as the primary; add natural variations.
